## 🛠️ Troubleshooting (Kai kažkas negerai)

### 🚨 Common Issues

#### 💀 "Permission denied"
```bash
chmod +x BadVPN.sh
sudo ./BadVPN.sh  # Force it!
```

#### 🔥 "systemctl not found"  
- Čekinki ar sistemoje yra systemd
- Ubuntu 15.04+ required (upgrade your system!)

#### ⚡ Installation fails
- Patikrink internet connection 🌐
- Run: `sudo apt update && sudo apt upgrade`
